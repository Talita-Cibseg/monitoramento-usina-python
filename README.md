# Monitoramento de Usina Solar ☀️

Projeto simples em Python para simular monitoramento de geração de energia solar.

## Funcionalidades
- Exibe geração atual
- Mostra meta diária
- Analisa desempenho da usina

## Tecnologias
- Python

## Objetivo
Projeto criado para prática de lógica de programação e monitoramento.
